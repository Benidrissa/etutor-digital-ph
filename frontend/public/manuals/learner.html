Manuel Apprenant

Bienvenue sur Sira — votre plateforme d'apprentissage adaptatif, bilingue et accessible hors ligne. Ce guide couvre toutes les fonctionnalités à votre disposition.

📱 Mobile-first 🇫🇷🇬🇧 FR / EN 🤖 Contenu IA 📶 Mode hors ligne

🎓1. Bienvenue sur Sira

Sira (du bambara Donniya Sira — « la voie du savoir ») est conçue pour les apprenants d'Afrique de l'Ouest. Elle génère automatiquement leçons, quiz et flashcards à partir de documents de référence.

📚 Formations

Suivre des cours, réviser avec des flashcards, passer des quiz adaptatifs, discuter avec un tuteur IA.

🌍 Contenu contextualisé

Les leçons et exemples s'adaptent à votre pays et à votre langue (FR/EN).

🤖 IA générative

Contenu généré par Claude (Anthropic) à partir des documents sources — unique et personnalisé.

📱 Offline

Téléchargez un module et continuez à apprendre sans connexion internet.

🔐2. Connexion & Inscription

Sira utilise une authentification sécurisée basée sur une application TOTP (Microsoft Authenticator, Google Authenticator).

Se connecter

  1. Saisissez votre email ou numéro de téléphone.
  2. Entrez votre mot de passe ou cliquez « Se connecter avec l'application d'authentification » pour un code TOTP.
  3. Cliquez Se connecter — vous serez redirigé vers le tableau de bord.

Récupérer l'accès (authentificateur perdu)

  1. Cliquez « Mot de passe oublié ? » sur la page de connexion.
  2. Saisissez votre email — un magic link valable 1 heure vous est envoyé.
  3. Cliquez le lien pour vous connecter et reconfigurer votre application.
💡
Inscription sur invitationLes inscriptions sont sur invitation. Cliquez « J'ai un code » sur la page de connexion si vous avez reçu un code d'activation.

📊3. Test de placement

Lors de votre première connexion, un test de placement de 20 questions évalue votre niveau de départ.

🎯 Objectif

Identification du niveau parmi 4 paliers : Débutant, Intermédiaire, Avancé, Expert.

⏱ Durée

Environ 20 minutes. Repassable après 3 mois.

🚀 Sauter un niveau

Cliquez « Tester mes acquis » sur un module verrouillé — score ≥ 85% débloque le module.

📈 Niveaux

Niveau 1 Débutant → Niveau 4 Expert

🏠4. Tableau de bord

Point d'entrée principal : progression globale, statistiques et modules de votre formation active.

🔥 Jours consécutifs

Votre streak quotidien. Connectez-vous chaque jour pour maintenir votre série.

📈 Score moyen

Score moyen sur tous les quiz — indicateur de maîtrise global.

🗓 Révisions dues

Flashcards à réviser aujourd'hui selon l'algorithme FSRS.

⚡ En cours

Modules commencés. Cliquez pour reprendre là où vous vous étiez arrêté.

📚5. Catalogue des formations

Liste toutes les formations publiées. Filtrez par domaine, niveau et type de public.

S'inscrire

  1. Cliquez sur une formation pour voir ses détails.
  2. Vérifiez modules, niveau requis et nombre de crédits (0 = gratuit).
  3. Cliquez S'inscrire ou Commencer. Si payant, vos crédits sont débités.
  4. Vous accédez directement au premier module.
⚠️
Formations payantesCertaines formations du marketplace expert sont payantes. Vérifiez votre solde de crédits dans Abonnement avant de vous inscrire.

📖6. Modules & Leçons

Chaque formation est divisée en modules, eux-mêmes composés d'unités (leçons, quiz, cas pratiques). Les modules se débloquent quand vous atteignez 80% sur le module précédent.

Structure d'une leçon

  • Introduction — Mise en contexte
  • Concept principal — Explication détaillée
  • Exemple contextuel — Cas réel adapté à votre pays
  • Termes clés — Glossaire FR/EN intégré
  • Sources — Références aux documents sources
💡
Contenu bilingueLe sélecteur FR/EN en haut à droite bascule la langue instantanément. Tout le contenu s'adapte.

✏️7. Quiz adaptatifs

Algorithme CAT (Computer Adaptive Testing) : la difficulté s'ajuste en temps réel selon vos réponses.

≥ 80%

Débloque le module suivant automatiquement.

≥ 85%

Débloque un module verrouillé via « Tester mes acquis ».

< 60%

Révision recommandée avant de réessayer.

CAT

Le test commence au niveau moyen, puis s'ajuste à votre niveau réel.

🃏8. Flashcards — Révision espacée

Algorithme FSRS (Free Spaced Repetition Scheduler) : révisions planifiées au moment optimal.

Comment réviser

  1. Accédez à Flashcards dans le menu.
  2. Lisez la question recto. Réfléchissez à la réponse.
  3. Cliquez pour révéler le verso.
  4. Évaluez votre souvenir : Nouveau / Difficile / Bien / Facile.
  5. L'algorithme planifie la prochaine révision selon votre évaluation.

🤖9. Tuteur IA

Assistant pédagogique propulsé par Claude (Anthropic) + RAG — répond en se basant sur les documents sources de votre cours, avec citations.

  1. Cliquez Tuteur IA dans la barre latérale.
  2. Posez votre question en français ou anglais.
  3. Les réponses citent les sources du cours.
  4. Le tuteur mémorise le contexte de vos échanges.
⚠️
Limite quotidienneL'usage du tuteur peut être limité selon votre abonnement. Vérifiez dans Abonnement.

🏆10. Certificats

Sira délivre un certificat de complétion quand vous terminez tous les modules et réussissez l'évaluation sommative.

  • Accédez à Certificats dans le menu — liste de vos certifications obtenues.
  • Chaque certificat indique : nom, formation, date, score final.
  • Téléchargement PDF ou partage d'un lien de vérification.

💳12. Abonnement & Crédits

Système de crédits (monnaie interne) pour accéder aux formations payantes. Abonnement SMS disponible (Orange Money, Wave).

Acheter des crédits

Via Paystack (carte bancaire, mobile money). Différents packages disponibles.

Utiliser des crédits

Pour s'inscrire à des formations payantes du marketplace expert.

Formations gratuites

Certaines formations sont gratuites (0 crédit).

Abonnement SMS

Orange Money ou Wave — accès illimité pour une durée définie.

📶13. Mode hors ligne

Sira est une PWA (Progressive Web App) conçue pour fonctionner sans connexion internet.

💡
Mise en cache automatiqueLes modules déjà ouverts sont mis en cache automatiquement. Pour télécharger explicitement un module, cliquez l'icône de téléchargement sur la page du module.

Ce qui fonctionne hors ligne

  • ✅ Lecture des leçons déjà consultées
  • ✅ Révision des flashcards
  • ✅ Navigation dans les modules téléchargés
  • ❌ Génération de nouveau contenu IA (nécessite une connexion)
  • ❌ Soumission de quiz (synchronisé à la reconnexion)

Installer l'application

  • Android Chrome — Menu (⋮) → « Ajouter à l'écran d'accueil »
  • iOS Safari — Bouton Partager (⬆) → « Sur l'écran d'accueil »
