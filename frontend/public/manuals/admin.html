Manuel Administrateur

En tant qu'Administrateur, vous avez un contrôle complet sur Sira — y compris les Paramètres système exclusifs à votre rôle.

🔑 Accès complet ⚙️ Paramètres système 👥 Tous les utilisateurs 📊 Analytics globaux

👑1. Rôle Administrateur

Le rôle admin est le niveau d'accès le plus élevé. Il inclut tout ce que peut faire un sous-admin, plus la page Paramètres de la plateforme.

✅ Tout ce que peut faire sub_admin

Utilisateurs, Formations, Curricula, Taxonomie, Groupes, Paiements, Analytique, Qualité, Journal d'audit.

⚙️ En plus : Paramètres

Configuration IA, sécurité, quiz et tous les paramètres système.

⚠️
ResponsabilitéLes changements de paramètres ont un impact immédiat sur tous les utilisateurs. Documentez tout changement dans le journal d'audit.

🗂️2. Navigation Admin

Accédez au panneau via Administration dans la barre latérale gauche. Les onglets du panneau donnent accès à toutes les sections.

👥3. Gestion des utilisateurs

Liste tous les comptes actifs. Recherche, création, modification de rôle, export CSV.

Créer un utilisateur

  1. Cliquez + Ajouter.
  2. Remplissez : nom, email, langue, pays, rôle.
  3. L'utilisateur reçoit un email d'activation avec magic link.

Rôles disponibles

  • Utilisateur — Apprenant standard
  • Expert — Crée des cours, gère des organisations
  • Sous-admin — Accès admin restreint (sans Paramètres)
  • Admin — Accès complet

📚4. Gestion des formations

Créez, publiez et archivez les formations. Chaque formation est générée par IA depuis des PDFs uploadés.

Créer une formation

  1. Cliquez + Créer une formation.
  2. Renseignez titre, description, domaine, niveau, audience.
  3. Uploadez les documents sources PDF/CSV/Word (indexés pgvector pour le RAG).
  4. Cliquez Générer la structure — l'IA produit le syllabus complet.
  5. Vérifiez puis Publiez.

Statuts

Brouillon

Visible admins uniquement. Leçons non générées.

Publié

Visible dans le catalogue aux apprenants inscrits.

Archivé

Retiré du catalogue. Inscriptions existantes maintenues.

RAG

PDFs → chunks 512 tokens → pgvector → Claude génère les leçons.

🗃️5. Gestion des curricula

Collection ordonnée de formations avec contrôle d'accès public/privé.

  1. Cliquez + Nouveau Curriculum.
  2. Titre, description, visibilité (public ou privé).
  3. Ajoutez des formations dans l'ordre.
  4. Pour curriculum privé, assignez des utilisateurs ou groupes.

🏷️6. Taxonomie

Catégories dynamiques (domaines, niveaux, types de public) administrées en base — aucune valeur codée en dur.

Modifier un domaine

  1. Cliquez ✏️ à droite du domaine.
  2. Modifiez le label FR et/ou EN.
  3. Sauvegardez — changement immédiat dans le catalogue.
⚠️
Supprimer un domaineVérifiez qu'aucune formation active ne l'utilise avant suppression.

👫7. Groupes d'utilisateurs

Gérez les accès aux curricula privés pour des ensembles d'utilisateurs.

  1. Cliquez + Nouveau groupe.
  2. Nommez le groupe (ex. « Cohorte 2026 »).
  3. Ajoutez membres par email ou importez CSV.
  4. Assignez ce groupe à un curriculum privé.

💰8. Paiements & Crédits

Toutes les transactions de crédits : achats, dépenses, gains experts, commissions, remboursements.

  • Achat — Achat via Paystack (carte, mobile money)
  • Inscription — Dépense pour s'inscrire à une formation payante
  • Revenu expert — 80% du prix lors d'une inscription à son cours
  • Commission — 20% part plateforme

📊9. Analytique

Vue globale de l'activité plateforme sur 7/30/90 derniers jours.

Total événements

Leçons lues, quiz passés, messages tuteur, etc.

Utilisateurs uniques

Utilisateurs avec au moins une activité.

Taux complétion quiz

% de quiz terminés sur le total démarré.

Modules populaires

Classement par nombre de consultations.

✅10. Qualité du contenu

File de validation des contenus IA. Contenu généré = statut « Brouillon » par défaut.

  1. Contenu généré → statut Brouillon.
  2. Apparaît dans la file Qualité.
  3. Approuver → publié aux apprenants. Rejeter → retourne en génération.
  4. Un apprenant accédant à du contenu non validé voit un avertissement.

📋11. Journal d'audit

Enregistre toutes les actions administratives : créations, modifications, suppressions, connexions.

ℹ️
Conformité réglementaireObligatoire : RGPD, Loi 2008-12 Sénégal, Ghana DPA, NDPR Nigeria. Conservation 3 ans minimum.

⚙️12. Paramètres de la plateforme

Exclusive au rôle Admin complet. Un sous-admin reçoit une erreur 403.

⚠️
Accès exclusif AdminNon accessible aux sous-admins. L'API retourne 403.

Catégories de paramètres

🤖 IA & Génération de contenu

  • Modèle Claude (Sonnet/Opus/Haiku)
  • Longueur cible des leçons
  • Température de génération
  • Validation automatique du contenu
  • Limite quotidienne de génération

🔐 Authentification & Sécurité

  • Mode inscription (ouvert / sur invitation)
  • Durée des JWT
  • Tentatives avant blocage
  • Durée des magic links

📝 Quiz & Évaluations

  • Score minimum déblocage (défaut 80%)
  • Score saut de niveau (défaut 85%)
  • Nombre de questions par quiz adaptatif
  • Délai re-tentative après échec
⚠️
Impact immédiatToute modification s'applique immédiatement à tous les utilisateurs. Documentez dans le journal d'audit.
